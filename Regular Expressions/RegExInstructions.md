# Copilot-Demos : Regular Expressions
## Example 1
Type //email regex in the regex.js file

Copilot should complete it for you with something like:

<pre>
const emailRegex = /^([a-zA-Z0-9_\-\.]+)@([a-zA-Z0-9_\-\.]+)\.([a-zA-Z]{2,5})$/;
</pre>
